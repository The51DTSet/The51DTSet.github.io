## 1. 작업 위치

- develop 브랜치에서 작업
- main 브랜치는 배포용

## 2. 패키지 설치

```shell
$ npm i
```

## 3. gatsby-cli 설치

```shell
$ (sudo) npm install -g gatsby-cli
```

## 4. 서버 띄우기

```shell
$ gatsby develop
$ gatsby develop -H 192.188.x.x
```

## 5. 배포하기

```shell
$ npm run deploy
```
